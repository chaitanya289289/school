Publisher Description



NSIS (Nullsoft Scriptable Install System) is an open source professional system developed to create Windows installers. With NSIS you can create installers capable of doing pretty much everything you need to setup your software. NSIS is script-based, allowing the user to handle even the most complex installation tasks. Numerous plug-ins and scripts are already available such as web installer creation, communication with Windows and other software components, shared components management.
